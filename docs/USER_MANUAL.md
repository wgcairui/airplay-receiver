# 📖 PadCast 用户手册

欢迎使用 PadCast！本手册将帮助您快速上手并充分利用 PadCast 的所有功能。

## 📱 应用介绍

PadCast 是专为 OPPO Pad 4 Pro (13.2英寸) 优化的 AirPlay 接收端应用，让您的平板轻松接收来自 Mac、iPhone、iPad 的无线投屏。

### 🎯 主要功能
- 📺 **无线投屏**: 接收 Mac/iPhone/iPad 的 AirPlay 投屏
- 🎵 **音视频同步**: 高精度同步，延迟低至 30ms
- ⚡ **硬件加速**: 支持 4K@30fps 高清播放
- 🔄 **智能监控**: 实时网络和性能监控
- ⚙️ **丰富设置**: 详细的音视频和网络配置

## 🚀 快速开始

### 第一次使用

1. **安装应用**
   - 从 [GitHub Releases](https://github.com/cairui/padcast/releases) 下载最新 APK
   - 在 OPPO Pad 上安装 APK 文件

2. **连接网络**
   - 确保 OPPO Pad 和投屏设备在同一 WiFi 网络
   - 建议使用 5GHz WiFi 网络以获得最佳性能

3. **启动应用**
   - 打开 PadCast 应用
   - 应用将自动开始监听 AirPlay 连接

4. **开始投屏**
   - 在 Mac/iPhone/iPad 上选择 AirPlay
   - 选择 "PadCast" 作为投屏目标
   - 享受高质量的无线投屏体验

- 🟡 **黄色**: 连接不稳定或性能警告

- ⚪ **灰色**: 未连接或服务未启动

### ⚙️ 设置配置

#### 🎥 视频设置

访问 **设置 > 视频设置**：

**质量选项**
- **低质量 (720p)**: 网络较慢时推荐
- **中等质量 (1080p)**: 平衡质量和性能
- **高质量 (1080p@60fps)**: 高刷新率体验
- **超高清 (4K)**: 最佳视觉效果（需要强劲网络）

**编解码器**
- **H.264**: 兼容性最好，适用于大多数设备
- **H.265/HEVC**: 更高压缩率，需要设备支持

**硬件加速**
- **启用**: 利用 GPU 解码，推荐开启
- **禁用**: 使用 CPU 解码，兼容性更好

#### 🎵 音频设置

访问 **设置 > 音频设置**：

**质量选项**
- **低质量 (128kbps)**: 节省带宽
- **中等质量 (256kbps)**: 平衡质量和带宽
- **高质量 (320kbps)**: CD 级音质
- **无损 (ALAC)**: 最佳音质体验

**音频增强**
- **数字信号处理**: 优化音频输出质量
- **动态范围压缩**: 平衡音量差异

**同步设置**
- **同步阈值**: 音视频同步的敏感度 (30-100ms)
- **缓冲大小**: 音频缓冲区大小设置

#### 🌐 网络设置

访问 **设置 > 网络设置**：

**连接模式**
- **自动检测**: 自动选择最佳网络
- **仅 WiFi**: 只使用 WiFi 连接
- **以太网优先**: 优先使用有线网络

**性能优化**
- **低延迟模式**: 减少缓冲以降低延迟
- **自动重连**: 网络中断时自动恢复
- **连接超时**: 连接超时时间设置 (5-30秒)

**高级选项**
- **端口设置**: 自定义 AirPlay 服务端口
- **mDNS 设置**: 设备发现服务配置
- **防火墙设置**: 网络访问权限配置

#### 🔧 高级设置

访问 **设置 > 高级设置**：

**性能监控**
- **启用监控**: 实时性能数据收集
- **监控频率**: 数据收集频率 (1-10秒)
- **历史数据**: 保存性能历史记录

**调试选项**
- **详细日志**: 启用详细的日志记录
- **性能分析**: 性能瓶颈分析工具
- **网络诊断**: 网络连接诊断工具

**实验功能**
- **测试模式**: 启用实验性功能
- **开发者选项**: 开发者专用设置

## 📱 投屏操作指南

### 从 Mac 投屏

1. **屏幕镜像**
   - 点击菜单栏的 AirPlay 图标
   - 选择 "PadCast" 设备
   - 选择 "镜像内置显示器"

2. **扩展显示**
   - 打开 "系统设置" > "显示器"
   - 点击 "添加显示器"
   - 选择 "PadCast" 作为扩展显示器

3. **视频播放**
   - 在支持的应用中播放视频
   - 点击 AirPlay 按钮
   - 选择 "PadCast" 设备

### 从 iPhone/iPad 投屏

1. **控制中心投屏**
   - 从右上角下拉打开控制中心
   - 长按 "屏幕镜像" 按钮
   - 选择 "PadCast" 设备

2. **应用内投屏**
   - 在支持 AirPlay 的应用中
   - 点击 AirPlay 图标
   - 选择 "PadCast" 设备

3. **音乐播放**
   - 在音乐应用中播放音频
   - 点击 AirPlay 音频图标
   - 选择 "PadCast" 设备

## 🔧 故障排除

### 常见问题

#### 无法发现 PadCast 设备

**可能原因**：
- 设备不在同一网络
- 防火墙阻止连接
- mDNS 服务未启动

**解决方案**：
1. 确认两设备在同一 WiFi 网络
2. 重启 PadCast 应用
3. 检查路由器设置，确保允许设备间通信
4. 关闭 VPN 或代理软件

#### 投屏延迟过高

**可能原因**：
- 网络带宽不足
- WiFi 信号较弱
- 设备性能不足

**解决方案**：
1. 切换到 5GHz WiFi 网络
2. 靠近路由器获得更好信号
3. 降低视频质量设置
4. 启用硬件加速
5. 关闭其他网络占用应用

#### 音视频不同步

**可能原因**：
- 网络抖动严重
- 音频缓冲设置不当
- 解码性能不足

**解决方案**：
1. 调整音视频同步阈值
2. 修改音频缓冲区大小
3. 启用低延迟模式
4. 重启投屏连接

#### 播放卡顿或掉帧

**可能原因**：
- 网络不稳定
- 设备过热
- 内存不足

**解决方案**：
1. 检查网络连接稳定性
2. 给设备降温，确保散热
3. 关闭其他运行中的应用
4. 降低视频质量或帧率

### 性能优化建议

#### 网络优化
- 使用 5GHz WiFi 网络
- 确保路由器支持 WiFi 6
- 减少网络中的其他设备活动
- 使用有线网络连接路由器

#### 设备优化
- 确保设备电量充足
- 关闭不必要的后台应用
- 定期清理设备存储空间
- 保持设备系统更新

#### 应用优化
- 定期更新 PadCast 应用
- 清理应用缓存数据
- 根据网络条件调整设置
- 启用自动化测试进行性能检测

## 📊 性能监控

### 实时监控指标

#### 📈 性能数据
- **帧率**: 当前视频帧率
- **延迟**: 端到端延迟时间
- **抖动**: 网络抖动情况
- **丢包**: 数据包丢失率

#### 💻 系统资源
- **CPU 使用率**: 处理器占用情况
- **内存占用**: 运行内存使用情况
- **网络带宽**: 上行/下行带宽使用
- **温度**: 设备运行温度

#### 🌐 网络状态
- **连接状态**: 网络连接稳定性
- **信号强度**: WiFi 信号质量
- **网络类型**: 当前使用的网络类型
- **IP 地址**: 设备网络地址信息

### 性能警告

应用会在检测到性能问题时发出警告：

- 🔴 **高延迟警告**: 延迟超过 150ms
- 🟡 **性能警告**: CPU 使用率超过 80%
- 🔵 **网络警告**: 网络不稳定或速度过慢
- ⚫ **系统警告**: 内存不足或设备过热

## 📋 常见问题 FAQ

### Q: PadCast 支持哪些设备？
A: 主要支持 OPPO Pad 4 Pro，同时兼容其他 Android 8.0+ 的平板设备。

### Q: 为什么投屏画质不清晰？
A: 检查网络带宽是否充足，尝试调高视频质量设置，确保使用 5GHz WiFi。

### Q: 投屏时有音画不同步怎么办？
A: 在音频设置中调整同步阈值，或启用低延迟模式。

### Q: 如何提高投屏的稳定性？
A: 确保网络稳定，启用自动重连，定期重启应用和设备。

### Q: PadCast 是否免费使用？
A: 是的，PadCast 是完全免费的开源应用。
